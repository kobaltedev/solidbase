# {frontmatter.title}

You can quickly create a new SolidStart project with SolidBase using the [with-solidbase](https://github.com/solidjs/solid-start/tree/main/examples/with-solidbase) template available in the Solid CLI.

## 1. Create a New Project

```package-create
solid@latest -s -t with-solidbase
```

## 2. Done!

Your project is now setup with SolidBase and ready to go!
