# Landing

The landing page is configured using [Frontmatter](/reference/frontmatter):

```md
---
layout: home

hero: ...

features: ...
---
```

## Required Setup

- set `layout: home`
- configure `hero`
- optionally configure `features`

## `hero`

Rendered by the [`Hero`](/reference/default-theme/components/hero) component.

Type:

```ts
type HeroConfig = {
  name?: string;
  text?: string;
  tagline?: string;
  image?: {
    src: string;
    alt?: string;
  };
  actions?: Array<{
    theme?: string;
    text?: string;
    link?: string;
  }>;
};
```

Notes:

- `title` still comes from frontmatter
- `hero.text` renders above the tagline
- `hero.tagline` renders with emphasized landing-page styling
- `hero.actions` renders CTA links
- external hero action links open in a new tab

## `features`

Rendered by the [`Features`](/reference/default-theme/components/features) component.

Type:

```ts
type FeaturesConfig = {
  icon?: string;
  title?: string;
  details?: string;
};
```

Notes:

- `icon` is rendered as HTML
- `title` renders as the feature heading
- `details` renders as the feature body copy
