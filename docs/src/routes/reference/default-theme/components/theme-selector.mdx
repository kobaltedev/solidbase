# {frontmatter.title}

Light, dark, and system theme switcher.

## Options

- `light`
- `dark`
- `system`

## Behavior

- uses Kobalte `Select`
- reads the current selection from `getThemeVariant()`
- persists changes with `setTheme()`
- refreshes its label on mount so client theme state replaces any server guess
