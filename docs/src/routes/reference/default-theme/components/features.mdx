# {frontmatter.title}

Landing-page feature grid.

## Props

```ts
function Features(props: {
  features: Array<{
    icon?: string;
    title?: string;
    details?: string;
  }>;
}): JSX.Element;
```

## Behavior

- creates one card per feature
- `icon` is injected as HTML
- `title` becomes the card heading
- `details` becomes the body copy
