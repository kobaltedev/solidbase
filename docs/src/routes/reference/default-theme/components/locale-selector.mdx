# {frontmatter.title}

Locale switcher for multi-locale sites.

## Data Source

- `useLocale().locales`
- `useLocale().currentLocale()`
- `useLocale().setLocale()`

## Behavior

- omitted when fewer than two locales are configured
- uses Kobalte `Select`
- displays each locale's `config.label`
- updates the active locale through `setLocale`
