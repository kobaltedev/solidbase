# {frontmatter.title}

Page table of contents.

## Data Source

- `useCurrentPageData()?.toc`

## Behavior

- omitted when no table of contents exists
- highlights the section nearest the current scroll position
- scrolls the active TOC item into view when needed
- clicking an item scrolls to the matching heading
- handles nested headings recursively
